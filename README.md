# Java Banking System

A Java-based banking system that supports multiple users with secure login, balance management, and deposit/withdraw operations.

## Features
- User account creation and authentication
- Deposit and withdrawal functionality
- Persistent user data stored in memory
- Object-oriented design using Java classes

## Technologies Used
- Java
- Object-Oriented Programming (OOP)

## How to Run
1. Clone the repository
2. Open the project in an IDE
3. Run `Bank.java`

## Future Improvements
- Transaction history tracking
- HashMap-based user storage
- Sorting users by balance
- JavaFX GUI
- Advanced data structures (BST, Heap)
